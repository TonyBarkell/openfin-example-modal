OpenFin - Links

Overview

OpenFin applies standard web behaviour when handling Anchour Links and respects any attributes applied to the Achour element. This example application explores the default behaviour and the options avaialble to control how links are handled, including:

  • Default Achor Tag behaviour
  • Opening Urls with the default web browser (using System.openUrlWithBrowser() )
  • Opening Urls with the default web browser (using System.launchExternalProcess() )
